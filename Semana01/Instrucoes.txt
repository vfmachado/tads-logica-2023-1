LÓGICA DE PROGRAMAÇÃO

LOGICA
- LÓGICA É UM TIPO DE RACIOCINIO
- DE ALGUMA FORMA ASSOCIADO A SEQUENCIA
- FORMA DE LIGAR OS PENSAMENTOS
- PARA SOLUCIONAR PROBLEMAS

PROGRAMAÇÃO
- SEQUENCIA DE COMANDOS / INSTRUCOES
- CRIA CÓDIGO PARA INSTRUIR A MAQUINA A FAZER ALGUMA COISA

LOGICA DE PROGRAMAÇÃO
- INSTRUCAO: "ORDEM"/COMANDO ISOLADO
- ALGORITMO: SEQUENCIA DE COMANDOS PARA SOLUCIONAR UM PROBLEMA

LINGUAGEM DE PROGRAMAÇÃO
- "É A MANEIRA" QUE FAZEMOS OS COMANDOS
- FORMA/FORMATO DOS COMANDOS

CÓDIGO FONTE

COMPILAÇÃO
- TRADUZIR DE CODIGO HUMANO PARA CODIGO DE MAQUINA

IDE - AMBIENTE DE DESENVOLVIMENTO INTEGRADO
- PROGRAMA QUE UTILIZAMOS PARA ESCREVER O CÓDIGO

TERMINAL - LINHA DE COMANDO

COMANDOS DO TERMINAL
pwd     mostrou a pasta atual
ls      lista o conteudo
cd P1   abre a pasta P1
